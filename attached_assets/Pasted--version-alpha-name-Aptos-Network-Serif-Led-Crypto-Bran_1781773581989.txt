## Colors

The palette uses 16 validated color tokens across 2 theme profiles. Semantic roles stay attached to observed usage so generation agents can choose accents without inventing new color meaning.

**Semantic naming:**
- **border-text** maps to `black`: Role "text" is grounded by usage context "Primary text, borders, nav items, hero heading — dominant foreground color across all zones".
- **surface-background** maps to `white`: Role "background" is grounded by usage context "Page background, nav bar background, off-white surface base".
- **content-background** maps to `blue`: Role "background" is grounded by usage context "Checkerboard pattern accent, decorative graphic element".
